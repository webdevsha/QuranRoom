Set Up QuranRoom

Connect your own Google Sheet — your data stays entirely yours. Takes about a minute.

1

Create a Google Sheet

Open a new spreadsheet at sheets.new (or reuse one you already have). This is where your daily logs will be saved.

2

Add the script copy & paste

In your sheet, go to Extensions → Apps Script, delete the placeholder code, and paste in the code below.

3

Deploy as a Web App

  1. Click Deploy → New deployment
  2. Click the gear icon next to "Select type" and choose Web app
  3. Set Execute as: Me and Who has access: Anyone
  4. Click Deploy, then authorize the script (click "Advanced" → "Go to project (unsafe)" → "Allow" — this warning is normal for personal scripts)
  5. Copy the Web app URL it gives you (starts with https://script.google.com/macros/s/...)
4

Paste your URL below

This saves it directly into the extension — nothing leaves your browser except the sync requests you send yourself.
